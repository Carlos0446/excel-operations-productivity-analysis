## Excel Operations & Productivity Analysis

This project analyzes labor cost, workload, and productivity across departments using Excel.

### Tools Used
- Microsoft Excel
- Pivot Tables
- Calculated KPIs

### Key Insights
- Operations accounts for ~42% of total labor cost and has the highest productivity score.
- Admin shows the lowest productivity score, indicating potential workflow inefficiencies.

This project demonstrates Excel-based reporting and analysis commonly used in entry-level analyst roles.
